📖 Guide du jeu

⚔ Unités

S
Soldat
⚔ ATK : 5
🛡 DEF : 4
👣 Déplacement : 1 case
Se déplace d'une case dans les 4 directions cardinales. Unité équilibrée, idéale pour tenir le territoire.
C
Cavalier
⚔ ATK : 4
🛡 DEF : 3
👣 Déplacement : 1 à 2 cases
Peut avancer de 1 ou 2 cases en ligne droite. Rapide pour capturer du territoire ou fuir.
T
Tank
⚔ ATK : 3
🛡 DEF : 6
👣 Déplacement : 1 case
🎯 Portée : Toute la carte
Peut attaquer n'importe quel ennemi non-adjacent. Chaque case de distance réduit son ATK de 1.

🎲 Système de combat

Chaque combat lance un dé D6 pour l'attaquant et un pour le défenseur.
🎲 Dé attaquant + ATK  vs  🎲 Dé défenseur + DEF
Si le total de l'attaquant est strictement supérieur, le défenseur est éliminé.
En cas d'égalité ou si le défenseur est plus haut, l'attaque échoue.
L'attaquant avance sur la case uniquement si tous les défenseurs ont été éliminés.
